### 1679. K 和数对的最大数目
给你一个整数数组 nums 和一个整数 k 。
每一步操作中，你需要从数组中选出和为 k 的两个整数，并将它们移出数组。
返回你可以对数组执行的最大操作数。
- 示例输入1: nums=[ 1,2,3,4],k=5
- 示例输出1: 2
- 示例解释1: 开始时, nums=[1,2,3,4]:移除1和4后, 之后是[2,3];再次移除2和3后, 之后是[];
不再有和为5的数对, 因此最后执行2次操作。
